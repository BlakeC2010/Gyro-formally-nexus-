# gyro AI — Test Prompts

Prompts to test every AI capability. Send each one and check the result.

---

## Basic Chat
1. **"What's 2+2?"** → Quick, correct answer
2. **"Write a 500 word essay on space exploration"** → Long streaming response, proper formatting

## Thinking Mode
3. **Toggle Thinking ON → "Is it more ethical to save 5 strangers or 1 loved one?"** → Live thinking panel shows reasoning, then collapses into final answer
4. **(Thinking OFF) "Compare quantum computing vs classical computing for cryptography, analyzing 3 different encryption methods"** → Thinking auto-enables for this complex query

## Deep Research
5. **Toggle Research ON → "What are the latest breakthroughs in fusion energy?"** → Plan card appears with Confirm / Cancel / Edit buttons (test all three)
6. **(Research OFF) "Do a comprehensive deep research analysis on the current state of AI regulation worldwide"** → AI auto-detects research intent, shows plan card for your confirmation first

## Web Search
7. **"Search the web for the latest SpaceX launch news"** → Response includes real, current web info

## Image Search
8. **"Show me pictures of the Northern Lights"** → Image carousel with thumbnails

## Memory
9. **"Remember that my favorite color is blue"** → "Remembered: ..." confirmation
10. **(New chat) "What's my favorite color?"** → Recalls "blue"

## Code & Artifacts
11. **"Write a Python web scraper that extracts headlines from a news site"** → Code block + clickable artifact card that opens in canvas

## Special Blocks
12. **"I want to learn a new language. Should I pick Spanish, Japanese, or Mandarin? Give me options"** → Interactive choice buttons appear
13. **"Make me a todo list for planning a road trip"** → Rendered checklist with checkboxes
14. **"Create a flowchart showing how a web request goes from browser to server and back"** → Visual Mermaid diagram

## Canvas
15. **Send prompt #11, open artifact in canvas, then type "Add error handling and logging" in the canvas input** → AI edits the code in-place

## File Handling
16. **Upload any text file → "Summarize this"** → AI reads and summarizes the file content

## Personalization
17. **"What do you know about me?"** → References your name, role, and any stored memories
